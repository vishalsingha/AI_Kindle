# Keyboard shortcuts

Every shortcut in AI Kindle. Mac uses `⌘`; Linux and Windows swap that for `Ctrl`.

[[toc]]

## Global

These work everywhere, even from inside text inputs or the command palette.

| Shortcut | Action |
|---|---|
| `⌘K` | Open the command palette |
| `⌘D` | Toggle dark / light theme |
| `Esc` | Back to library (from the reader) |

## Reader

Available when a book is open.

### Navigation

| Shortcut | Action |
|---|---|
| `Space`, `↓`, mouse wheel | Scroll down |
| `Shift+Space`, `↑` | Scroll up |
| `PageDown` | Jump to next page |
| `PageUp` | Jump to previous page |
| `⌘→` | Next page |
| `⌘←` | Previous page |
| `Home` | First page |
| `End` | Last page |

### Zoom

| Shortcut | Action |
|---|---|
| `⌘+` | Zoom in |
| `⌘-` | Zoom out |
| Trackpad pinch | Zoom around cursor |
| `Ctrl+scroll` | Same as pinch |
| Click the % in the bottom bar | Reset zoom |

### Panels

| Shortcut | Action |
|---|---|
| `⌘B` | Toggle sidebar (TOC + annotations) |
| `⌘J` | Toggle AI panel |
| Click the notebook icon | Toggle notes panel |

## Library

| Shortcut | Action |
|---|---|
| Type any letter | Focus the search bar |
| `Enter` on a book card | Open it |
| `⌘`-click a book | Toggle that book into bulk selection |
| `Shift`-click a book | Range-select up to that book |
| `Esc` | Clear selection |

## Import dialog

When a duplicate or title-collision dialog appears:

| Shortcut | Action |
|---|---|
| `Enter` | Confirm the import with the displayed title |
| `Esc` | Skip this file |

## Annotations toolbar

After selecting text:

| Click | Action |
|---|---|
| Yellow / Green / Blue / Pink / Orange swatch | Highlight in that color |
| Speech bubble icon | Open the comment popover |
| Pencil icon | Create an inline text note |
| Sparkle icon (AI configured) | Summarize selection |
| Brain icon (AI configured) | Explain selection |

## Command palette

Within `⌘K`:

| Type | What you get |
|---|---|
| Any word | Fuzzy match across book titles and built-in actions |
| `:123` or "page 123" (while a book is open) | Jump to that page |
| Plain digits (while a book is open) | Same — direct page jump |
| `↑` / `↓` | Move through results |
| `Enter` | Confirm |
| `Esc` | Close palette |
